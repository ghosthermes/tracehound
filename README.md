# tracehound
Adversarial API testing framework.  Consumes OpenAPI schemas and automatically generates edge-case requests.

Tracehound turns API documentation into adversarial test cases. Feed it a schema and it generates malformed, boundary, and authentication-focused requests designed to expose implementation drift.
